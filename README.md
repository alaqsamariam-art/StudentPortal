#  اسم المشروع
🎓 (Smart Student Portal)بوابة الطالب الذكية  
---
 
##  وصف المشروع 
بوابة الطالب الذكية هي نظام ويب يهدف إلى تسهيل الخدمات الطلابية من خلال تمكين الطلاب من إنشاء حساب، وتسجيل الدخول،وتسجيل المساقات بنفسه، وتقديم الشكاوى ومتابعة حالتها إلكترونيًا، بينما يستطيع المسؤول إدارة المستخدمين والشكاوى والرد عليها وتحديث حالتها من خلال لوحة تحكم خاصة، وعرض التقارير الاحصائية، وعرض ال ER_Diagram.
---

## 💡 فكرة المشروع
**بوابة الطالب الذكية** هي تطبيق ويب ديناميكي متجاوب (Dynamic Responsive Web Application) يتيح للطلبة إنشاء حسابات جديدة، تسجيل الدخول، وتسجيل المساقات المتطابقة مع تخصصاتهم. كما يوفر قسماً فريداً للشكاوى يتيح للطالب إرسال استفساره مباشرة للدعم الفني مع متابعة الردود الواردة إليه أولاً بأول مصحوبة بالإشعارات التنبيهية الفورية عند حل المشكلة.

 على الجانب الآخر، يمتلك النظام **لوحة إدارة مخصصة (Admin Dashboard)** تتيح للمسؤولين إدارة وتعديل حسابات الطلاب، مراجعة وتصفية الشكاوى الواردة حسب التصنيف والحالة، وتقديم الردود الرسمية لحلها وعرض التقارير الاحصائية، وعرض ال ER_Diagram..
